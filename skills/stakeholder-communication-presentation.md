# Skill — Stakeholder Communication & Presentation

> التواصل مع الأطراف المعنية والعرض

| Field | Value |
|---|---|
| **Category** | Behavioral |
| **Assessment method** | THREE_SIXTY_EVALUATION |
| **Applies to** | Business Development & External Contracting; SP and above |

## Description

Communicating with influence — presenting to clients, partners and executives, and adapting the message to persuade and align stakeholders.

## Proficiency Levels

| Level | Label | What it means for this skill |
|---|---|---|
| 1 | Awareness | Communicates clearly in routine, one‑to‑one settings |
| 2 | Basic | Delivers prepared updates and simple presentations |
| 3 | Intermediate | Presents confidently to groups and tailors content to the audience |
| 4 | Advanced | Influences senior stakeholders; handles tough rooms and Q&A |
| 5 | Expert | Sets the communication standard; represents the company in high‑stakes forums |

## Assessment Methods

Configured inline on the skill (**Admin › Skills › Competency Standard › Assessment Methods**) and conforms to **360° Multi-Rater, BARS, SHRM BASK**.

| Setting | Value |
|---|---|
| Method | 360° Multi-Rater Evaluation |
| Rater weights | Self 10% / Peer 30% / Manager 60% |
| Assessor (manager rater) | Direct Manager |
| Frequency | Any time within the year (annual) |
| Audience | All employees assigned the skill |

**Behaviourally-anchored criteria** (rated 1–5, weighted to 100):

- Communicates clearly and adapts the message to the audience. — **35%**
- Presents confidently to groups and handles tough Q&A. — **35%**
- Influences and aligns senior stakeholders. — **30%**

## Notes

Behavioral skill — evidenced via 360° feedback from clients, peers and managers.

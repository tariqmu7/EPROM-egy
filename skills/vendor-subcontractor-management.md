# Skill — Vendor & Subcontractor Management

| Field | Value |
|---|---|

## Description

Selecting, on-boarding and managing vendors, contractors and subcontractors — managing performance, relationships and contractual obligations to secure reliable, value-for-money external delivery.

## Proficiency Levels

| Level | Label | What it means for this skill |
|---|---|---|
| 1 | Awareness | Knows key vendors / subcontractors and basic supplier processes |
| 2 | Basic | Maintains vendor records and handles routine follow-up with guidance |
| 3 | Intermediate | Manages assigned vendors / subcontractors and monitors their performance |
| 4 | Advanced | Owns strategic vendor / subcontractor relationships and drives performance and value |
| 5 | Expert | Sets vendor-management strategy, segmentation and performance standards |

## Assessment Methods

Configured inline on the skill (**Admin › Skills › Competency Standard › Assessment Methods**) and conforms to **360° Multi-Rater, BARS, SHRM BASK**.

| Setting | Value |
|---|---|
| Method | 360° Multi-Rater Evaluation |
| Rater weights | Self 10% / Peer 30% / Manager 60% |
| Assessor (manager rater) | Direct Manager |
| Frequency | Any time within the year (annual) |
| Audience | All employees assigned the skill |

**Behaviourally-anchored criteria** (rated 1–5, weighted to 100):

- Selects, onboards and manages vendors / subcontractors effectively. — **35%**
- Monitors performance against SLAs and resolves issues constructively. — **35%**
- Maintains fair, compliant and value-driven supplier relationships. — **30%**

## Notes

The relationship side of external delivery — parallels Client Relationship Management but faces the supply side. Evidence: vendor scorecards, performance reviews, qualification records.
